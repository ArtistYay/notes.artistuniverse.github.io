1. What does the Azure Security Center investigation graph encompass?
    
    A specific entity and the entities that are related to it.
    

1. In the Azure Portal, you configure app service authentication so that all anonymous requests to `/.auth/login/<provider>` are redirected to the chosen provider. You receive an anonymous request from a built-in mobile app. What is the most likely returned response to the built-in mobile app?
    
    HTTP 200 OK
    

1. Which feature do you use to encrypt data both at rest and in motion in an Azure SQL database?
    
    Always Encrypted
    

1. When should you run a Microsoft Azure Security Playbook in Security Center?
    
    When you need to respond to a specific security alert.
    

1. Which operating systems are supported by Microsoft Monitoring Agent?
    
    Windows and Linux VMs
    

1. In a Platform-as-a-Service (PaaS) model for Microsoft Azure, which of the following responsibilities is transferred to Microsoft?
    
    Applying security patches to the operating system.
    

1. You are developing an application you want to connect a Windows VM to a Linux VM. Which protocol will you use and what key should you use for this?
    
    Secure Shell protocol and SSH keys
    

1. What Azure service could you use to provide inbound protection at the perimeter?
    
    Azure Application Gateway
    

1. What is the name of the load balancer service from Azure that includes a web application firewall and is used to protect HTTP traffic?
    
    Azure Application Gateway
    

1. You are reviewing Azure Security Center recommendations. A vulnerability exists within a network security group. Which category is this vulnerability?
    
    High Severity
    

1. What protocol is used by Azure Active Directory to authorize access to web apps and web APIs?
    
    OAuth 2.0
