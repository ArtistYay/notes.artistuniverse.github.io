- **"exact phrase"**
    - Use double quotes.
    - Finds pages with the _exact_ wording.
    - Example: `"passive reconnaissance"`
- **`site::`**
    - Limits search to a specific domain.
    - Example: `site:tryhackme.com success stories` (Finds success stories on TryHackMe)
- **`-` (minus sign)**
    - Excludes results containing a word/phrase.
    - Example: `pyramids -tourism` (Finds pyramid info, but not tourism sites)
- **`filetype::`**
    - Finds specific file types.
    - Examples:
        - `filetype:pdf cyber security` (PDFs about cyber security)
        - `filetype:ppt cyber security` (PowerPoint presentations about cyber security)
        - Other file types: `.doc`, `.xls`
- **Advanced Controls**
    - Many more operators exist.
    - Check specific search engine documentation.
    - Resource: [Advanced search operators list](https://github.com/cipher387/Advanced-search-operators-list?tab=readme-ov-file)
