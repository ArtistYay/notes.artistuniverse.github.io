- You can create and use VMs in the cloud.

- It provides IaaS.

- Scale sets allow you to centrally manage, configure, and update a large number of VMs in minutes to provide highly available applications. It’s like AWS auto-scale but it’s not.

- Azure Batch enables large-scale parallel and high-performance computing (HPC) batch jobs with the ability to scale to tens, hundreds, or thousands of VMs.
